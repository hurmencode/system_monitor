# 🖥 System Resource Monitor

Локальный монитор ресурсов системы для Linux, написанный на **C++ и Python**.

Проект состоит из:
- **C++-ядра**, которое собирает системные метрики
- **Python-клиента**, который отображает данные пользователю

---

## 🚀 Возможности

- Отображение:
  - загрузки CPU
  - использования RAM
  - uptime системы
- Список топ процессов по CPU
- Обновление данных в реальном времени
- Чёткое разделение логики и UI
